# Ai-Voice-Organizer


Это Telegram-бот для управления задачами, напоминаниями и идеями с поддержкой голосового ввода и интеграцией с LLM.

Проект позволяет пользователю отправлять голосовые или текстовые сообщения, после чего система:

* распознаёт речь,
* определяет тип запроса,
* извлекает нужные данные,
* сохраняет их в базу данных,
* показывает задачи в Telegram,
* отправляет напоминания и уведомления по расписанию.

Основные возможности

* Парсинг задач, напоминаний, идей и дедлайнов через LLM
* Поддержка голосовых сообщений через локальный Speech-to-Text
* Работа с датой и временем с учётом часового пояса Europe/Moscow
* Человекочитаемое форматирование дат для сообщений в Telegram
* Получение активных задач пользователя через jOOQ
* Отображение задач по одной в Telegram
* Inline-кнопки под задачами:
    * ✅ отметить как выполненную
    * 🗑 удалить
* Обработка callback-запросов от кнопок
* Защита сценария во время незавершённого диалога с пользователем
* Выделенный UpdateRouter для маршрутизации входящих обновлений
* Scheduler, который раз в минуту проверяет:
    * напоминания по задачам
    * обычные напоминания
* Отправка уведомлений в Telegram
* После отправки:
    * для задач выставляется reminder_sent = true
    * обычные напоминания удаляются из базы

Стек

* Java 17
* Spring Boot
* PostgreSQL
* Liquibase
* jOOQ
* TelegramBots
* LM Studio
* Whisper / локальный STT

Идея проекта

Проект сделан как умный Telegram-органайзер, который позволяет пользователю управлять личными делами более естественным способом — через обычную речь и короткие текстовые команды, без ручного заполнения форм.
